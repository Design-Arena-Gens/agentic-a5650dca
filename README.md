# Expense Compass

Interactive personal expense dashboard built with Next.js, React, Tailwind CSS, and Recharts. The app lives in the `dashboard` workspace directory and was designed for quick deployment to Vercel.

## Getting Started

```bash
cd dashboard
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the dashboard.

## Available Scripts

- `npm run dev` – starts the development server
- `npm run build` – builds the production bundle
- `npm start` – serves the production build
- `npm run lint` – runs ESLint

## Features

- Month selector with comparative insights
- Budget vs. actuals category breakdown
- Multi-month spending trend visualization
- Searchable, filterable transaction table
- Upcoming bills tracker

## Tech Stack

- Next.js (App Router, TypeScript)
- React 19
- Tailwind CSS
- Recharts
- Lucide Icons
